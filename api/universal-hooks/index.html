Plugin Hooks

Init

Called when a plugin is being initialized. Other plugins may or may not be present, dependent on load order.
Return behavior: No return behavior.

copy 

private void Init()
{
    Puts("Add commands, check config file, register messages or permissions, etc.");
}

Loaded

Called when a plugin has finished loading. Other plugins may or may not be present, dependent on load order.
Return behavior: No return behavior.

copy 

private void Loaded()
{
    Puts("Handle stuff after plugin is initailized such as referencing another plugin");
}

Unload

Called when a plugin is being unloaded.
Return behavior: No return behavior.

copy 

private void Unload()
{
    Puts("Save files, destroy timers, etc.");
}

LoadDefaultConfig

Called when the config for a plugin should be initialized. Only called if the config file does not already exist.
Return behavior: No return behavior.

copy 

private new void LoadDefaultConfig()
{
    Puts("Default configuration created");
}

OnFrame

Called by engine every server frame.
Return behavior: No return behavior.

copy 

private void OnFrame()
{
    Puts("Handle with care!");
}

OnPluginLoaded

Called when a plugin has been loaded.
Return behavior: No return behavior.

copy 

private void OnPluginLoaded(Plugin name)
{
    Puts($"Plugin '{name}' has been loaded");
}

OnPluginUnloaded

Called when a plugin has been unloaded.
Return behavior: No return behavior.

copy 

private void OnPluginUnloaded(Plugin name)
{
    Puts($"Plugin '{name}' has been unloaded");
}

Server Hooks

OnServerInitialized

Called after the server startup has been completed and is awaiting connections.
Return behavior: No return behavior.

copy 

private void OnServerInitialized()
{
    Puts("Grab game information after startup");
}

OnServerSave

Called before the server saves and shuts down.
Return behavior: No return behavior.

copy 

private void OnServerSave()
{
    Puts("A good time to save plugin data!");
}

OnServerShutdown

Called after plugins are unloaded and before the server shuts down process.
Return behavior: No return behavior.

copy 

private void OnServerShutdown()
{
    Puts("The server is shutting down, goodbye!");
}

Player Hooks

CanPlayerConnect

Called when a player is attempting to connect to the server.
Return behavior: Return true/false to approve/reject.

copy 

private bool CanPlayerConnect(string name, string id, string ip)
{
    Puts($"Player {name} ({id}) at {ip} is attempting to connect...");
    if (ip == "127.0.0.1") return false;
    return true;
}

OnPlayerApproved

Called after a player is approved to connect to the server.
Return behavior: No return behavior.

copy 

private void OnPlayerApproved(string name, string id, string ip)
{
    Puts($"Player {name} ({id}) at {ip} has been approved to login!");
}

OnPlayerBanned

Called when a player is banned by the server, the game developer, EAC, or VAC.
Return behavior: No return behavior.

copy 

private void OnPlayerBanned(string name, string id, string ip, string reason)
{
    Puts($"Player {name} ({id}) banned for {reason ?? "Unknown"} - {ip}");
}

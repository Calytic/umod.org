Compiling Source

While we recommend using one of the official releases, you can compile your own builds if you'd like. Keep in mind that only official releases are supported by the uMod team and community. Good luck!

    Downloading

    Download a Git client of your choice such as GitHub Desktop or SourceTree if you do not already have a Git client installed and ready.

  • 3

    Setup

    Download and install Visual Studio 2015 or 2017 (community is free, but any edition will work) if you do not already have it installed.
  • 4

    Building

    Open the uMod.sln solution file in Visual Studio 2015 or 2017. Once opened, set the target to Release and click Build. If you get errors, you're likely not using the latest Visual Studio (2015 or 2017) which is required as uMod uses some C# 6.0 features.
  • 5

    Deploying

    Copy the files from the Bundles directory to your game server installation, then start the server! To save you a step in the future, simply add the path to a .deploy file under the desired game extension directory such as Games/Unity/UMod.Game.Rust for automatic deployment.

Introduction

While legacy plugins are no longer supported by uMod for public plugin submissions, it is still possible to install the legacy plugins if your host allows it.

Getting started

  • 1

    Download the extension and dependencies

    JavaScript & CoffeeScript Download
    LUA Download
    Python Download
  • 2

    Completely shutdown the server

    Ensure the server is offline in order to replace any existing files

  • 3

    Move or upload the above files to the location where uMod is currently installed

    The location of a uMod installation will depend on which game server it is installed on. Search for "uMod.Core.dll" and that is likely the correct location.
    Examples: RustDedicated_Data/Managed for Rust, Hurtworld_Data/Managed for Hurtworld, etc.

  • 4

    Start the server

    Now the server will be able to load JavaScript (.js), CoffeeScript (.coffee), LUA (.lua), and/or Python (.py) plugins depending on which of the above extensions are installed

Notes

The above languages are no longer bundled with uMod due to lacking interest and dedicated support for the language. There are very few public plugins written in these language, but the extensions will be maintained as necessary for basic plugin functionality.

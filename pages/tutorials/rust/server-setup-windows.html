Downloading Steam CMD

SteamCMD is a command-line client for Steam that can install and update the Rust dedicated server files.
Download SteamCMD
Extract and run SteamCMD. The first time SteamCMD runs, it will update itself to the latest version. Allow SteamCMD to update and then proceed to the next step.

Downloading the dedicated server

After opening SteamCMD, authentication is required using either valid Steam credentials or an anonymous account. Enter the following to begin anonymous authentication.
copy 
login anonymous
Next, choose the directory where your Rust dedicated server installation will be placed by entering the following
copy 
force_install_dir
Finally, begin the Rust dedicated server installation by entering the following
copy 
app_update 258550
Please be patient as the Rust dedicated server installation may take a few minutes to download depending on your internet connection speed.
SCREENSHOT 2 PLACEHOLDER

Basic install/update script

The previous steps may be automated by created a batch (.bat) file which runs the following script

copy 
@steamcmd +login anonymous +app_update 258550 validate +quit
pause

Adding uMod to the server

Download the latest uMod build that is compatible with your server version first. Extract uMod to the folder where RustDedicated.exe is located and overwrite when prompted. This will add the required uMod files and the modified Rust files which will load uMod.

Starting the server

Create a batch (.bat) file in your Rust dedicated server folder and enter the following script

copy 
@echo off
RustDedicated.exe -batchmode +server.port 28015 +rcon.port 28016 +rcon.ip 0.0.0.0 +rcon.password "changeme"

Launch the batch file

SCREENSHOT 3 PLACEHOLDER

Joining the server

Launch the Rust game client and open the in-game console by pressing the F1 key. Then enter the following and press the Enter key

copy 
connect 127.0.0.1:28015

To make the server accessible to other players, the external IP address must be obtained and all necessary port forwarding rules must be configured on the firewall(s).

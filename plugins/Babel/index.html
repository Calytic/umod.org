Babel

Plugin API for translating messages using free or paid translation services

Babel is a simple, easy-to-use plugin API for translating messages using free or paid translation services.

Supported Services

Google (free)

  • Limits: Not sure the exact figure, but it’s pretty limited
  • API key: None! This isn’t really an API, it’s just scraping a webpage

Google (paid)

  • Limits: 2 million characters per day and 100,000 characters per 100 seconds
  • API key: You can get an API key by creating an application on the console
  • Bonus: Trial + $300 credit to spend on Google Cloud Platform over 60 days
  • https://cloud.google.com/translate/

Microsoft (v1 API only, free and paid)

  • Limits: 2 million characters per month on free tier, see other pricing tiers
  • V1 API: These keys you cannot get anymore, so you’ll need an existing one
  • V2 API: Get an API key by creating an application here - not available yet
  • https://www.microsoft.com/en-us/translator/translatorapi.aspx

Yandex (free and paid)

  • Limits: 10 million characters per month on free tier and 1 million characters per 24 hours, see pricing
  • API key: Get an API key by signing up here
  • https://tech.yandex.com/translate/

Configuration

You can configure the settings in the Babel.json file under the oxide/config directory.

{ "API key (if required)": "", "Translation service": "google" }

Available translation service options are “google”, “microsoft”, and “yandex”.

Plugin Developers

To call the methods provided by this plugin, your plugin needs to get the plugin reference. You can do this a couple different ways depending on if it optionally depends on it or requires it.

If your plugin requires this plugin and is written in C#, you can use the following at the top of your plugin:

// Requires: Babel

For C# plugins, you’ll need to get the plugin’s reference as such:

[PluginReference] private Plugin Babel;

Now that you have the plugin reference, you call the provided methods:

Babel?.Call("Translate", "Hello world!", "en", "auto");

Note: The above examples are not using the optional Action callback parameters. For examples of how to use these, see Babel Chat or other plugins that use this plugin.

API Methods

Translate(string message, string to, string from = "auto", Action<string> callback = null)

Translates text from one language to another another

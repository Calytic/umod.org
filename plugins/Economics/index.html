Basic economics system and economy API

Economics is a basic economics system that other plugins can make use of the provided API. On its own, this plugin only provides the features listed and has no shop.

Permissions

This plugin uses Oxide’s permission system. To assign a permission, use oxide.grant user <name or steam id> <permission>. To remove a permission, use oxide.revoke user <name or steam id> <permission>.

  • economics.use – allows player to use basic economic commands
  • economics.admin – allows player to use admin commands

Commands

  • balance - check your balance
  • balance - *check [name] player balance*
  • deposit - *deposit [amount] to [name] player*
  • setmoney - *set [amount] balance for [name] player*
  • transfer - *transfer [amount] to [name] player*
  • withdraw - *withdraw [amount] from [name] player*
  • ecowipe - wipes all saved Economics data

Configuration

You can configure the settings in the Economics.json file under the oxide/config directory.

“Remove unused accounts (true/false)”: true – On server start, players with the default balance are removed

“Start money amount (any number 1 or above)”: 1000 – Amount of money a player starts with

“Money transfer fee (any number 0.00 or above)”: 0.01 – Fee for money transfers (0.01 = 1%), 0.00 to disable

Localization

The default messages are in the Economics.json file under the oxide/lang/en directory. To add support for another language, create a new language folder (ex. de for German) if not already created, copy the default language file to the new folder, and then customize the messages.

Commands

  • balance – check your balance

  • deposit <name> <amount> – deposit [amount] to [name] player

  • withdraw <name> <amount> – withdraw [amount] from [name] player

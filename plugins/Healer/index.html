Healer

Allows players with permission to heal themselves or others

  • Permissions
  • Commands
  • Configuration
  • Localization
  • Credits

Healer allows players with permission to heal themselves or others. Also cures, hydrates, etc. when used in Rust. By default, there is a cooldown of 30 seconds, but it can be disabled by setting it to 0 in the configuration file.

Permissions

This plugin uses Oxide’s permission system. To assign a permission, use oxide.grant user <name or steam id> <permission>. To remove a permission, use oxide.revoke user <name or steam id> <permission>.

  • healer.others – allows player to use the ‘heal’ command to heal others
  • healer.self – allows player to use the ‘heal’ command to heal themselves
  • healer.all – allows player to use the ‘healall’ command to heal everyone

Commands

  • heal <amount>
  • heal <name or id>
  • heal <amount> <name or id>
  • healall
  • healall <amount>

Replace <amount> with a valid number, and <name or id> with a player’s name or Steam ID

Configuration

You can configure the settings in the Healer.json file under the oxide/config directory.

“Cooldown in seconds (0 to disable)”: 30 – Number of seconds a player must wait to use a command again

“Maximum heal amount (1 - infinity)”: 100 – The maximum amount that can be used with the commands

Localization

The default messages are in the Healer.json file under the oxide/lang/en directory. Language files are also available for French, German, Russian, and Spanish by default. To add support for another language, create a new language folder (ex. de for German) if not already created, copy the default language file to the new folder, and then customize the messages.

"CommandUsage": "Usage: {0} <amount> <name or id> (target optional)", "Cooldown": "Wait a bit before attempting to use '{0}' again", "NotAllowed": "You are not allowed to use the '{0}' command", "PlayerNotFound": "Player '{0}' was not found", "PlayerWasHealed": "{0} was healed {1}", "PlayersHealed": "All players have been healed {0}!", "YouWereHealed": "You were healed {0}"

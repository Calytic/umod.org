Plugins

Expand your server functionality with one or many plugins

AdminChatroom

Allows admins to chat privately with each other in-game

AdminPanel

A small GUI panel of clickable commands for admins

AdminRadar

Allows admins to have a radar to help detect cheaters

AdvancedUpgrading

Advanced upgrading that disallows players from upgrading building blocks quickly

Analytics

Real-time collection and reporting of server data to Google Analytics

Babel

Plugin API for translating messages using free or paid translation services

BabelChat

Translates chat messages to each player's language preference or server default

BanDeleteBy

Removes all entities placed by a player when they get banned

BetterChatFlood

Adds a cooldown to chat to prevent flooding

BodiesToBags

Instantly turns player corpses into backpacks

BuildingRestriction

Restricts building height, building in water, and number of foundations

C4Logger

Logs useful information about the use of C4

ChatCleaner

Clears/resets a player's chat when joining the server and on command

Checkpoints

Restore points for players to teleport to when the server restarts

ClanBan

Ban an entire clan with one command

CommandBlock

Blocks configured commands from being executed on the server

CupboardMessages

Sends a configured message to a player when they place a tool cupboard

DangerousTreasures

Dangerous event with treasure chests

DevTest

Tests all of the available Oxide hooks and provides useful helpers

DiscordMessages

Sends report and ban messages straight to Discord

DropRemover

Removes selected world items from the map with commands

Duelist

1v1 and TDM dueling event with plugin integrations

Economics

Basic economics system and economy API

EnhancedBanSystem

Ban players by Steam ID, IP address, local ban list, or remote ban list

Everlight

Allows infinite light from configured objects by not consuming fuel

ForceEmote

Force another player to do an emote on command

FortWars

Custom gamemode that changes values of gathering/smelting/crafting based on a current phase

Freeze

Stops one or more players on command and keeps them from moving

HandyMan

AOE repair functionality to those with permission

ItemTranslations

Provides a data file of item names for translation

KarmaKills

Rewards players on karma on kill, or takes karma away from them

Logger

Configurable logging of chat, commands, connections, and more

NoCraft

Blacklist craft items or disable crafting altogether

NoRaid

Prevents players from destroying buildings they're not associated with

NoSuicide

Stops players from suiciding/killing themselves

Password

Provides name and chat command password protection for the server

PhantomSleepers

Create phantom sleepers to lure ESP hackers

Portgun

Teleports players with permission to object or terrain they are looking at

RaidTracker

Log raids in progress by tracking explosions to be later recreated on-screen

RealtimeWipeInfo

Auto title and description updating, auto wipe schedule, chat filter, and more

Respawner

Automatically respawns players with permission and optionally wakes them up

Robbery

Players can steal money, points, and/or items from other players

ServerAnnouncer

Allows you to broadcast a message to the server with a custom prefix

StartMachine

Automatically start machines on server startup and by manual control

StartupCommands

Automatically runs configured commands on server startup

StashGap

Sets the distance at which players must build from stashes

SteamGroups

Automatically adds members of Steam group(s) to a permissions group

SupplySignalAlerts

Broadcasts a message when someone throws a supply signal

TwitchAuth

Only allow your followers on Twitch to join your server

Underworld

Teleports admins/developer under the world when they disconnect

Vanish

Allows players with permission to become truly invisible

WelcomeTP

Teleports new players to a set position

Whisper

Message in-game players from the server console or RCON

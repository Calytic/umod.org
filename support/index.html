Contributing

Reporting Bugs

Whenever you encounter something that doesn't work as it should, we'd love it if you could file a bug report on PLACEHOLDER.

Make sure to include a detailed description of the issue and the steps needed for reproducing it. Attaching a small test case makes this even easier for us. Get detailed instructions on how to file a good bug report in our bug reporting guide.

Security Issues

Please disclose security issues responsibly by emailing security@umod.org with a full description. We'll work on releasing an updated build as quickly as possible. Please do not email non-security issues; use the forums or issue tracker instead.
